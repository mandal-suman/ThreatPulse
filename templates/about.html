About ThreatPulse

Your centralized source for cybersecurity news and updates

Our Mission

ThreatPulse aggregates the latest cybersecurity news from multiple trusted sources, providing security professionals, researchers, and enthusiasts with a single platform to stay informed about threats, vulnerabilities, breaches, and industry developments.

Features

Real-time Updates

Source Filtering

Filter news by specific sources to focus on what matters

REST API

Access articles programmatically via our JSON API

Responsive Design

Works seamlessly across desktop, tablet, and mobile devices

Technology Stack

  • Backend: Python, Flask
  • RSS Parsing: feedparser library
  • Frontend: HTML5, CSS3, JavaScript
  • Icons: Font Awesome

API Documentation

The following API endpoints are available:

GET /api/articles

Retrieve all articles in JSON format

Parameters: source (optional), limit (optional)

GET /api/sources

Get list of all available news sources

GET /api/refresh

Force refresh the articles cache
